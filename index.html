CSS is Cascading Style Sheet and a way to style your website through a different type of language. To implement CSS you need to add a link in your html to the CSS file. InLine CSS refers to changing the style of a tag in the line where the tag is located by adding attributes. Internal CSS refers to using the style tag in the html code space to add attributes and this overrides the external CSS. External CSS is a code space in an css file where changes to the style can be made but this can be overridden by InLine CSS or Internal CSS.
